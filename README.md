# HR Data Analytics Dashboard 📊

## Project Overview
This project is an HR Data Analytics Dashboard created using Power BI.  
The dashboard helps analyze employee attrition, salary distribution, job satisfaction, department performance, and workforce trends.

The main objective of this project is to convert raw HR data into meaningful business insights for better decision-making.

---

## Tools & Technologies Used
- Power BI
- Power Query
- DAX
- Excel
- Data Visualization

---

## Key Features
✅ Employee Attrition Analysis  
✅ Department-wise Employee Count  
✅ Gender Diversity Analysis  
✅ Salary Distribution Insights  
✅ Job Satisfaction Tracking  
✅ Interactive Filters & KPI Cards  

---

## Key Insights
- Identified departments with high attrition rate
- Compared employee satisfaction levels
- Analyzed salary trends across departments
- Tracked workforce demographics
